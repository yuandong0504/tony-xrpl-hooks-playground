# XRPL Hooks Playground

A simple CLI simulator for testing XRPL Hooks logic locally.

## Goal
Help XRPL developers prototype and debug Hooks logic before deployment.

## Current Status
MVP v0.1

## Files
- `simulator.py`: basic CLI simulator prototype

## Roadmap
- add transaction simulation
- add hook result inspection
- add example hooks
- support XRPL test scenarios

## Vision
This project aims to become a lightweight open-source developer tool for XRPL Hooks experimentation and onboarding.
